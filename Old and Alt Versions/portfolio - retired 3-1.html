Leveraged Javascript, HTML, and CSS to visualize various sorting algorithms. An array is randomly generated and displayed as vertical bars, the user can adjust the size of the array and speed of the algorithm being visualized. Each step of the algorithm from start to finish is highlighted through color changing and moving bars.

Included algorithms: bubble, selection, insertion, quick, merge.
